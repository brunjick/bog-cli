# bog-cli
🇬🇪 CLI tool for Bank of Georgia

⚠️ This is just a proof of concept, no real usage options implemented (yet)

## Installation
```bash
git clone -b pre-alpha https://github.com/brunjick/bog-cli.git
cd bog-cli
npm i
npm i -g
```

## Usage

##### Login
``` bog login [username] ```

##### Logout
``` bog logout ```

##### Get authenticated user info
``` bog info ```
